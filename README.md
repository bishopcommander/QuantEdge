# 🚀 QuantEdge

**QuantEdge** is a modern quantitative trading simulation platform that enables users to design, test, and analyze stock-specific trading strategies in a clean, risk-free environment.

It focuses on making quant trading **accessible, visual, and experimentation-driven** — without involving real money.

---

## 📌 What is QuantEdge?

QuantEdge is built for:
- 🧑‍💻 Developers exploring quant finance  
- 📊 Students learning trading strategies  
- 📈 Traders testing ideas before real execution  

It provides a **simulation-first workflow** where users can go from idea → strategy → backtest → insights seamlessly.

## ⚙️ Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/bishopcommander/QuantEdge.git
cd QuantEdge
```

---

### 2. Run Frontend
```bash
cd frontend
npm install
npm run dev
```

---

### 3. Run Backend
```bash
cd backend
./mvnw spring-boot:run
```
